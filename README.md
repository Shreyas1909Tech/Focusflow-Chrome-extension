# Focusflow-Chrome-extension
FocusFlow is a full-stack productivity tracking tool built as a Chrome Extension with a MERN (MongoDB, Express, React, Node.js) backend. instantly, and review your daily and weekly habits through detailed reports.
Installation & Setup:
1. Clone the Repository
git clone https://github.com/your-username/focusflow.git
cd focusflow
2. Setup Backend
cd backend
npm install
npm start
3. Setup Frontend (Dashboard)
cd frontend
npm install
npm start
4. Setup Chrome Extension
Open Chrome and go to: chrome://extensions/
Enable Developer Mode
Click Load Unpacked
Select the extension folder
5. Run the Application
Backend runs on: http://localhost:5000
Frontend runs on: http://localhost:3000
Open Chrome and start using the extension
